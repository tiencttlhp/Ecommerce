### Download

1. Ấn vào button "Code"

2. Chọn Download ZIP để tải file Zip về máy

3. Giải nén file Zip

### Setup

1. VS Code
   1. Mở Folder vừa giải nén trên
   2. Mở Terminal ((<kbd>CTRL</kbd> + <kbd>`</kbd> by default))
   3. Install node module cho server
      * Gõ `npm install`

2. Khác
